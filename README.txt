WordBattler4.1

●起動方法
1.このファイルがあるディレクトリ内で、「python manage.py runserver ポート番号」を実行
（例）python manage.py runserver 80

2.Google Chromeで「localhost:ポート番号」にアクセスする。（1,2のポート番号は同じ数字）
（例）localhost:80

●遊び方
ソロモード
1.トップページより「ソロモード」を選択し、名前を入力して「決定」を押す
2.画面中央に敵が出現し、お題に対して意味の近い「単語」を入力し、攻撃する
3.敵の体力をすべてなくすと、次のステージへ進める（全10ステージ）
（補足）条件として、制限時間やターン数がある
4.ゲームオーバーや敵の体力をすべてなくしたあと出てくる画面で「ホームに戻る」ボタンを押すと、ランキングに記録される

協力モード
1.トップページより「協力モード」を選択し、名前と入る部屋を選択する
2.二人人数が集まると自動的にゲームがスタートする
3.ゲームの進行方法はソロモードと同じ
